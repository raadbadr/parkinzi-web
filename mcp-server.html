MCP Server ● LIVE

اربط Claude أو ChatGPT أو أي مساعد ذكاء اصطناعي يدعم MCP بقاعدة بيانات PARKINZI الحية. ابحث عن المواقف، الشواحن الكهربائية، وإحصائيات المنصة مباشرة من المحادثة.

الاتصال بالـ Server

Endpoint: https://parkinzi.com/mcp

Transport: streamable-http · Auth: none · Read-only.

كيف تربطه في Claude

  1. افتح claude.ai/settings/connectors أو في تطبيق Claude Desktop: Settings → Connectors.
  2. اضغط Add custom connector.
  3. الصق الرابط: https://parkinzi.com/mcp
  4. اضغط Add، ثم فعّل الـ connector.
  5. في أي محادثة، اطلب من Claude: "ابحث عن مواقف PARKINZI قرب موقعي".

كيف تربطه في ChatGPT

  1. افتح ChatGPT (نسخة Plus أو Pro أو Enterprise).
  2. Settings → Connectors → Add custom connector.
  3. الصق نفس الرابط: https://parkinzi.com/mcp
  4. احفظ، ثم في المحادثة فعّل الـ connector وابدأ الاستخدام.

الخادم يطبّق عقد search / fetch الذي يتطلبه ChatGPT، لذلك يعمل أيضاً في وضع Deep Research — اطلب بحثاً معمّقاً عن المواقف في السعودية وسيستشهد ببيانات PARKINZI ومقالات المدونة كمصادر.

الأدوات المتاحة

search

بحث موحّد (ChatGPT contract): مواقف + مدونة + صفحات.

fetch

جلب النص الكامل لأي نتيجة بحث بمعرّفها.

search_parking_spots

بحث المواقف بالموقع، المسافة، أو النص الحر.

find_ev_chargers

شواحن السيارات الكهربائية قرب موقع.

get_spot_details

تفاصيل كاملة لموقف معين.

get_platform_stats

إحصائيات المنصة الحية.

list_supported_cities

المدن والمناطق المغطاة.

search_blog_posts

بحث في مقالات المدونة.

get_app_info

معلومات تحميل التطبيق وأرقام الدعم.

أمثلة طلبات

بعد ما تربط الـ MCP في Claude، جرّب:

  • "ابحث لي عن موقف فيه شاحن كهربائي قرب الرياض بارك."
  • "كم موقف PARKINZI متوفر حالياً في جدة؟"
  • "لخّص لي آخر 3 مقالات في مدونة PARKINZI عن السيارات الكهربائية."
  • "اعطني إحصائيات PARKINZI الحالية: عدد المواقف، الشواحن، المدن."

للمطوّرين

الـ server يستخدم JSON-RPC 2.0. مثال طلب يدوي:

curl -X POST https://parkinzi.com/mcp \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'

مثال تنفيذ أداة:

curl -X POST https://parkinzi.com/mcp \
  -H 'Content-Type: application/json' \
  -d '{
    "jsonrpc": "2.0",
    "id": 2,
    "method": "tools/call",
    "params": {
      "name": "find_ev_chargers",
      "arguments": {
        "latitude": 24.7136,
        "longitude": 46.6753,
        "radius_km": 5
      }
    }
  }'

الـ discovery manifest متاح على:

GET https://parkinzi.com/.well-known/mcp.json

الحدود والاستخدام

  • الوصول مجاني وبدون مصادقة (read-only).
  • القراءة فقط — لا يدعم الحجز أو الدفع حالياً (مرحلة لاحقة).
  • Rate limiting على مستوى Cloudflare WAF لتفادي الإساءة.
  • جميع البيانات حية من قاعدة بيانات PARKINZI الإنتاجية.

الخصوصية

الـ MCP server لا يكشف بيانات المستخدمين. يكشف فقط بيانات المواقف العامة (نفس البيانات التي تظهر على الخريطة في الصفحة الرئيسية). راجع سياسة الخصوصية للتفاصيل.
